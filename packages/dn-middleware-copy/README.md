## 文件复制中间件
在不需要编译，而仅需要将文件按规则从一个地方复制到别一个地方时，可使用 copy 中间件

### 示例

如下示例，将所有 md 文件复制到 build 目录，并通过使用 (1) 保留上一级目录结构

```yml
build
  - name: copy
    files:
      ./build/(1)/(0).md: ./**/*.md
```

`files` 配置节的 `value` 为源文件匹配表达式，`key` 为目标文件生成表达式，`(n)` 为路径中某级的名称，`0` 为当前文件名称，「从右向左」依次为各级目录名称。
